# React Pizza Menu

A simple React application built while learning the fundamentals of React development.

The project focuses on:

- Components
- Props
- JSX
- Rendering lists with map()
- Conditional rendering

## Features

- Display pizza menu dynamically from an array of objects
- Reusable React components
- Conditional rendering for available/sold out pizzas
- Dynamic opening hours
- Clean and simple UI

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS

## What I Learned

Through this project, I practiced:

- Creating and using React components
- Passing data using props
- Writing JSX syntax
- Rendering arrays using `map()`
- Using conditional rendering
- Organizing React project structure
